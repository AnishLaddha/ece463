## Overview

This repository contains projects, assignments, and notes created for the ECE 463: Computer Networks class at Purdue University. The course provides an in-depth introduction to the design and implementation of computer communication networks. It emphasizes the fundamental design principles and concepts that have led to the success of the global Internet.

## Course Description

ECE 463 explores a range of topics critical to understanding and developing computer networks:

- **Digital Transmission and Multiplexing**: Basics of data transmission over various media and techniques to combine multiple signals.
- **Protocols**: Rules governing data communication, including HTTP, DNS, and more.
- **MAC Layer Design**: Principles of Media Access Control, including Ethernet and 802.11 (Wi-Fi) standards.
- **LAN Interconnects and Switching**: Techniques for linking and managing local area networks, including switches and routers.
- **Congestion, Flow, and Error Control**: Mechanisms to manage data traffic, prevent congestion, and ensure data integrity.
- **Routing and Addressing**: Methods for determining data paths and assigning addresses in networks.
- **Performance Evaluation**: Techniques for assessing and optimizing network performance.
- **Internetworking (Internet)**: Detailed study of the Internet architecture, including TCP/IP protocols.
